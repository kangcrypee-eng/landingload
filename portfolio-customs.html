ERP MODULES

관세사 특화 5대 기능 모듈

일반 통관부터 중고차 수출 통관까지, 관세사 업무 전체를 하나의 시스템으로 통합합니다.

📋
일반 면장(통관) 관리
일반 수출입 통관 면장을 등록하고 진행 상태를 실시간으로 추적합니다.
  • 면장 입력·조회·현황 통합
  • 접수 → 처리중 → 세관제출 → 통관완료 단계 관리
  • 수출타입·목적국·출발지·반입지 관리
  • 월별 면장 현황 집계
  • 엑셀 다운로드 지원
🚗
중고자동차 면장 관리
중고차 수출 전용 면장 관리. RORO·컨테이너 수출 타입별로 처리합니다.
  • 차대번호·등록번호·차종 기반 조회
  • RORO·컨테이너 수출타입 구분
  • 목적국·출발지·반입지 관리
  • 업체별 담당자 배정
  • 면장 현황 대시보드
🏢
거래처 관리
일반·중고자동차 거래처를 분리 관리하고 협력사 정보를 통합합니다.
  • 일반 거래처 · 중고자동차 거래처 분리
  • 법인·개인·외국인 유형 구분
  • 사업자번호·대표자·연락처 관리
  • 거래처별 면장·말소 건수 집계
  • 중단/해지 업체 관리
💰
청구서 관리
말소·면장 청구서를 월별로 집계하고 거래처별로 발급합니다.
  • 말소 청구 출력·현황
  • 면장 청구 현황 통합
  • 월별 청구·입금·미수금 집계
  • 일괄 청구 생성
  • 엑셀·카카오톡 공유
📊
월별 현황 파악
면장 건수·청구금액·거래업체 수를 월별로 한눈에 파악합니다.
  • 총 면장 건수·총 청구금액·거래 업체 수
  • 월별 12개월 현황 그리드
  • 연도별 데이터 비교
  • 수출타입·목적국별 분류
  • 엑셀 다운로드
